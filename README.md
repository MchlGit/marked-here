# Marked Here: Spots worth a pause.

Marked Here is a full-stack web app that allows users to share and rate scenic and accessible local mini landmarks. Users can share whatever they discover on their outdoor walks whether it be a tree in bloom, a colorful rock garden, or a randomly placed soda machine.

Live Demo -> https://www.markedhere.com 

---

## Features 

- **Create a spot:**
  Users can create a Spot using their current location or by placing a marker on a map.
- **Browse spots:**
  Users can view all Spot listings.
